Creating a test is simple. First, launch the built in test editor. One can create a new test or load a previous one. There are two main sections of the test, the head and the questions. 


The head stores data about the test as a whole.  Below is a list of the settings and what they do:


Setting Name

Description and Possible Values

Required?

ENABLERANDOMIZATION

Sets whether the question order is to be randomized for each student. (1 or 0)

No

ENABLEUSERNAMES

Specifies whether or not user names are to be displayed when looking at results of individual questions. (1 or 0)

No

DRIVESUBFOLDER

Specifies which shared drive folder the answer files generated by the students will be placed in. It will be created if it does not exist. (Text)

Yes

CTMANAME

The name of the corresponding answer key file. (Text)

Yes

DRIVEFOLDERROOT

The name of the folder that was shared with the students on Google Drive. (Text)

Yes

TESTNAME

This is simply the name of the test to be displayed in the title bar during test taking and results viewing. (Text)

Yes

APPENDPERIOD

Specifies whether or not the period number will be appended to the folder name upon the student's uploading of their answer files. (1 or 0)

No

WORDBANK

Contains a comma separated list of words to be used in word bank styled questions.

No



The next section are the questions. Each question has its own settings as well.  These are listed below:


Setting Name

Description and Possible Values

Required?

QUESTIONTAG

Tag of the question (Text)

No

QUESTIONNUMBER

The number of the question to be displayed during test taking and results viewing. (Number)

Yes

QUESTIONSTYLE

The type of question.  (WORDBANK, MULTIPLECHOICE or FREERESPONSE)

Yes

QUESTIONTITLE

The actual question itself. (Text)

Yes

ANSWERCHOICE

This sets up what answer choices the students have. Each answer choice is to be set per setting. Multiple ANSWERCHOICE settings can be present. These settings are read in order. (Text)

Only for MULTIPLECHOICE style questions.

IMAGE

Specifies a corresponding image to be displayed during the test. The entire file name including extension must be stated. (Text)

No

MAXANSWERS

Specifies how many answers the students may select. This is presumed to be 1 if not listed. (Number)

No



NOTE: Question number is not editable directly through the test editor. Question numbers are automatically applied on save. One may use the arrows next to the questions to change the order. 


NOTE: Questions with a style of Word Bank or Free Response cannot have answers applied to them. They may appear added but they will not be saved. Use the word bank setting in the test head in order to provide answers for word bank questions.


After the test has been created and saved, one must generate an answer key. This can be done by simply taking the test yourself. Use this time to also check for any errors. Save the answers locally. 


NOTE: Free response questions will not be graded when the results are displayed. 


NOTE: Make sure to take the CTMA file out of the directory where the rest of the answers are stored, otherwise it will included with the results.
