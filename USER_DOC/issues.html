I get a "Main Class Not Found" error while trying to launch the application
Your current version of Java is out of date. Java 7 or later is needed to the this application.

When taking a test, going to a question with an image freezes the program.
Java is very slow loading images, especially from media that is not your hard drive. Try to resize your image so it is not large. This should cut down on the loading times.

Images won't load properly
Check to make sure the image path is correct. You can do this by accessing the debug console. On the main screen, click on advanced then Debug Console.

When editing test questions, question numbers are off.
Question numbers are resorted when the test is saved.
